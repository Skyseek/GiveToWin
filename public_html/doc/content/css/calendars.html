Calendar classes

All month-calendars use table as a structure. There are some common classes you can use to style cells within these:

  • week-end
  • other-month
  • today
  • unavailable

Week calendar

The following features are available:

Easy positioning

Use the following classes to define an event position within a day:

  • from-7-30, from-7-45, from-8, from-8-15,... to from-20-15 to define event start
  • to-7-30, to-7-45, to-8, to-8-15,... to to-20-15 to define event end
Example: <li class="from-8 to-12"></li>, <li class="from-9-45 to-10-15"></li>...
Multiple event colors
  • Grey (default)
  • Blue (use class event-blue)
  • Green (use class event-green)
  • Orange (use class event-orange)
  • Purple (use class event-purple)
Dot events
For whole day events
Simultaneous events
Display side by side
Break hours display
Use the class lunch : <li class="lunch from-12 to-14"></li>
Visual style for unavailable user/resource
Add the class unavailable to the event list: lt;li class="unavailable">
