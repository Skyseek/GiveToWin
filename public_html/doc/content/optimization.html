Optimization

About template optimization

The Constellation Admin Skin uses many CSS3 effects, such as gradients or drop-shadows, which can be heavy on older browsers if your pages are very long. To improve rendering speed, you may remove some of these CSS3 effects, using the guide below.

How to optimize?

To easily remove unwanted CSS code, run the following regular expression in your favorite IDE and replace them with nothing:

Remove drop-shadows
\s+(-moz-|-webkit-)?(box|text)-shadow:[^;]+;
Remove gradients
\s+background: *(-moz-linear|-webkit)-gradient[^;]+;
Remove background sizing
\s+(-o-|-moz-|-webkit-)?background-size:[^;]+;
Remove empty selectors
[\.\-_#:a-zA-Z0-9,\s ]+\{\s*\}
