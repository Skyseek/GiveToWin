Merging an update with your own changes

If you made changes in the CSS code or the JS code before downloading an update, you can use WinMerge (for Windows) or TextWrangler (for Mac) to compare your own files an the ones from the updated package, spot the differences and choose which one to keep. It is quite fast and really easy!
